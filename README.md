# Découpe FEC - Outil gratuit pour couper les fichiers FEC trop volumine's

Fichier FEC trop gros pour votre logiciel comptable ?  
Cet outil découpe votre fichier FEC en autant de parties que vous voulez, en gardant l’en-tête intact.

### Fonctionnalités
- Interface graphique simple (aucune installation)
- Choix du nombre de fichiers à générer
- Nommage automatique : `MonFichier_part1.fec`, `MonFichier_part2.fec`, etc.
- Fonctionne sur **Windows 10/11** sans Python
- 100 % gratuit et open-source

### Comment l'utiliser ?
1. Téléchargez la dernière version (Release) ou le ZIP ci-dessous  
2. Dézippez le dossier n’importe où  
3. Double-cliquez sur `Découpe FEC.exe`  
4. Sélectionnez votre fichier FEC → choisissez le nombre de parties → c’est fini !

### Téléchargement direct (dernière version)
[**Télécharger Découpe FEC (Windows)**](https://github.com/Dbouhet/decoupe-fec/releases/latest/download/Decoupe.FEC.zip)

### Code source
Le code est dans le dossier `/src`. Vous pouvez le modifier ou le recompiler avec PyInstaller si besoin.
